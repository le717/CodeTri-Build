# CodeTri-Build #
> A custom-built, Node/`npm` script-powered build system.

## Features ##
* Basic file copying
* SCSS build system
  * libsass
  * Autoprefixer
* JavaScript build system
  * UglifyJS 2
    * Minify/concatenate
    * Removes IE8 optimizations
    * Drops `console.*` functions
